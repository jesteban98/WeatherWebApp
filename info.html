How do you use this web application?

To make use of the map you have to click on it and move the cursor, searching the city whose forecast you are looking for. Then, an icon with the representation of the weather will load. Click on it and you will get the current data of that place.

If you want to search the name of a city instead, type the city name and, optionally, type the country code (ES,PT,GB,FR,...).After clicking the submit button a chart willl appear showing the temperature for the next 5 days every 3 hours.

To change the units, just click on the button corresponding to the units you want to use. The next query will show the data in those units.
